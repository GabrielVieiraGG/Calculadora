Projeto de uma calculadora básica.

A calculadora conta com as quatro operações básicas e o uso de parênteses. 

Os comandos devem ser inseridos via terminal como forma de expressões matemáticas, como no exemplo:

2+3
2-5*4
-3+(3*6)

São reconhecidos somente algarismos (0 a 9), vírgula ou ponto para separar decimais e parênteses. Caso outro símbolo seja inserido a calculadora retorna erro.
